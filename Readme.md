Smart Accessibility is an AI-powered web application designed to enhance digital inclusivity for people with disabilities by integrating multiple intelligent assistive features into a single platform. The system enables real-time sign language detection using a trained deep learning model, optical character recognition (OCR) to extract text from images, voice command functionality for hands-free interaction, live weather updates based on user location, geolocation detection, and email integration for easy communication. Developed using a Flask-based backend and an interactive frontend interface, the project focuses on creating a user-friendly, accessible, and intelligent digital environment that empowers differently-abled individuals to interact with technology more independently and efficiently.
